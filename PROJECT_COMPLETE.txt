🎯 SYSTEM FEATURES
═══════════════════════════════════════════════════════════════════════════

Registration Flow:
  1. 👤 USER SIGNS UP
     └─ Submits email, password, role, file upload
     └─ Status: PENDING (awaiting admin approval)

  2. ✅ ADMIN APPROVES
     └─ Reviews application documents
     └─ Generates 6-digit code (15-min expiry)
     └─ Sends verification email via SMTP
     └─ Status: APPROVED (awaiting email verification)

  3. ✔️ USER VERIFIES
     └─ Receives email with 6-digit code
     └─ Enters code on verification form
     └─ Status: VERIFIED (fully activated)

📡 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════

POST /register
   ├─ Input: email, password, fullName, role, phone, files
   ├─ Validation: Email format, password strength, file type/size
   └─ Output: 201 Created with user_id

POST /admin/approve-user
   ├─ Input: user_id, action (approve/decline)
   ├─ Actions: Generate code, send email, log action
   └─ Output: 200 OK with expiry timestamp

POST /verify
   ├─ Input: email, verification_code (6 digits)
   ├─ Validation: Code format, match, expiry
   └─ Output: 200 OK, user fully verified
